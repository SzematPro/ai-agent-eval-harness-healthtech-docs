:::caution[Documentação de referência: não é um dispositivo médico]
Este site é uma **referência de capacidades e prontidão**, não uma certificação de conformidade nem aconselhamento jurídico. O sistema é construído e avaliado com **dados 100% sintéticos**; **não é um dispositivo médico**, não é clinicamente validado e não manipula **PHI de produção**.
:::

Esta é a referência pública de uma implementação de referência que demonstra **engenharia de IA generativa orientada à medição para setores regulados**: um agente conversacional de saúde para adesão à medicação cuja afirmação clínica sempre deve citar um cartão verificado da base de conhecimento, acompanhado de um arcabouço de avaliação validado por CI que o comprova.
