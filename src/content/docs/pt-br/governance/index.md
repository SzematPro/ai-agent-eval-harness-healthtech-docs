# Governança e prontidão

Esta seção mapeia a implementação de referência
`ai-agent-eval-harness-healthtech` para os marcos regulatórios, os regimes de
proteção de dados e os controles de segurança operacional que regem os sistemas
de IA em healthtech. Cada documento é uma avaliação de prontidão ou um plano de
controle, não uma certificação, e toda afirmação se restringe ao sistema tal
como avaliado com dados sintéticos.
