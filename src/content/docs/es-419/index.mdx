:::caution[Documentación de referencia: no es un dispositivo médico]
Este sitio es una **referencia de capacidades y preparación**, no una certificación de cumplimiento ni asesoría legal. El sistema se construye y evalúa con **datos 100% sintéticos**; **no es un dispositivo médico**, no está validado clínicamente y no maneja **PHI de producción**.
:::

Esta es la referencia pública de una implementación de referencia que demuestra **ingeniería de IA generativa orientada a la medición para industrias reguladas**: un agente conversacional de salud para la adherencia a la medicación cuya afirmación clínica siempre debe citar una tarjeta verificada de la base de conocimiento, junto con un arnés de evaluación validado por CI que lo demuestra.
