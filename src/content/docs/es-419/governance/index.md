# Gobernanza y preparación

Esta sección mapea la implementación de referencia
`ai-agent-eval-harness-healthtech` a los marcos regulatorios, los regímenes de
protección de datos y los controles de seguridad operativa que rigen los
sistemas de IA en healthtech. Cada documento es una evaluación de preparación o
un plan de control, no una certificación, y toda afirmación se acota al sistema
tal como se evaluó con datos sintéticos.
